<h1>Chat<h1>

<h3>SOBRE:</h3>

Olá Devs!
Aqui está um Mini Projetinho desenvolvido usando "Vue 3 e Laravel".
<br>
O sistema se trata de um Chat em tempo real.<br>
<br>

<h3>FERRAMENTAS:</h3>
Na criação do sistema utilizei as seguintes ferramentas:

* VUE 3 (Front-end)<br>
* LARAVEL (Back-end)<br>
* Autenticação JSON Web Token (JWT)<br>
* SWEETALERT<br>
